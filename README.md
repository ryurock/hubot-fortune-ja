hubot-fortune-ja
================

Returns fortune information from http://fortune.yahoo.co.jp/12astro/ranking.html

## Installation


* Run the ```npm install``` command

```
npm install hubot-fortune-ja
```

* Add the following code in your external-scripts.json file.

```
["hubot-fortune-ja"]
```

## Usage

```
bin/hubot

hubot 占い
```
